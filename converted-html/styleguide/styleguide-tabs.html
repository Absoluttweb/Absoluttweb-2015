SCSS Tabs 
Tabs kommer uten styling, og man må lage custom class [.tabs-*] i style.scss. Dette er ett eksempel:
.tabs-1 li {
	a {
	border-bottom: 2px solid $light-grey-color;
	}
	&.active a {
		border-color: $secondary-color;
		color: $header-color;
	}
	&:focus a, &:hover a {
		color: $header-color;
	}
}

Kode Readmore 
Integrer JS Readmore Plugin. CSS er allerede inkludert i base.scss

DOM:
<div class="readmore">
<p>...</p>
</div>

Aktiver med følgende skript:
<script>
//Readmore
$(".readmore").readmore({
embedCSS: false,
maxHeight: 72,
moreLink: '<a href="#">...vis all tekst</a>',
lessLink: '<a href="#">skjul tekst</a>'
});
</script>
